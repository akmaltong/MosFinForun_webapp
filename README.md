# МФФ 3D Навигация

Интерактивное веб-приложение для 3D-навигации по Московскому Финансовому Форуму. Построено на React + Three.js с поддержкой AR.

## Возможности

- 3D-визуализация площадки форума с baked-освещением
- Навигация по зонам (конференции, выставки, еда, регистрация, лаунж)
- Поиск маршрутов через NavMesh (three-pathfinding)
- Режимы камеры: вид сверху, под углом, от первого лица
- AR-режим
- Мини-карта
- Расписание мероприятий с уведомлениями
- Отслеживание друзей на карте
- Настройки графики (качество, тональная компрессия, освещение)

## Стек

- React 19 + TypeScript
- Three.js / React Three Fiber / Drei
- Zustand (стейт-менеджмент)
- Tailwind CSS
- Vite

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Структура проекта

```
src/
├── components/    # React-компоненты (3D-сцена, UI-панели, маркеры)
├── data/          # Моковые данные и POI
├── hooks/         # Кастомные хуки
├── store/         # Zustand store
├── styles/        # Стили панелей
├── types/         # TypeScript типы
└── utils/         # Навигация, коллизии, NavMesh
public/
├── *.glb          # 3D-модели (площадка, NavMesh, зоны)
└── textures/      # Текстуры окружения и площадки
```
